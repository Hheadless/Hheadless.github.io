    # 弹性盒

    1. 开启弹性盒 display:flex
    2. 改变弹性子元素的排列方向
    flex-direction: row  /* 横向从左到右，默认排列 */
                row-reverse  /* 反转横向，丛右到左 */
                column   /* 纵向排列   常用*/
                column-reverse  /* 反转纵向 */
    1. 弹性内容沿主轴线的对齐方式 (开启弹性盒后 横向的轴线 不会根据排列方向而改变。)
    justify-content: flex-start  /* 弹性内容向行头挨着排列，默认值 */
                 flex-end     /* 弹性内容向行尾紧挨着排列 */
                 center    /* 弹性内容居中挨着排列   常用*/
                 space-between  /* 平均分布在一行，两端靠边   常用*/
                 space-around   /* 平均分布在一行，两端留空间，中间的空隙事两端空间的一半   常用*/
    1. 弹性内容沿主轴线的对齐方式 (开启弹性盒后 纵向的轴线 不会根据排列方向而改变。)
    /* 和上面的一个意思 */
align-items: flex-start  
             flex-end
             center    /* 常用 */
             baseline    /* 几乎不用 */
             stretch    /* 几乎不用 */
    1. 弹性内容的换行方式
    flex-wrap:  nowrap   /* 超出不换行    常用*/
            wrap   /* 换行显示      常用*/
            wrap-reverse   /* 反转 */
